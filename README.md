# algorithm

## 定位

本项目用于分享典型的、优质的算法题。
解法不限语言，更偏重于解题思路。

对于每道题：

至少包含：
- 题目描述（QD.md => QuestionDescription）
- 解题思路（外部链接也可）
- 代码示例

可包含：
- 测试用例
- 其他

可提供公共的工具方法，比如对于链表、树等结构的打印、生成工具等。

## 组织形式

可参考维度：
- leetcode题目号
- 题目所属分类

由于此项目不作为leetcode题解查询，更偏重于典型题目的集合，因此不建议走题目号索引。
